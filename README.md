#  Smart Fishing Rod – IoT Project

##  Description
This is an academic IoT project developed as part of my Computer Science studies.
The system uses sensors to monitor marine conditions and helps identify suitable fish species.

##  Technologies
- ESP32
- Temperature sensor
- Salinity sensor
- Water current sensor
- Android application (Java)
- SQLite database

##  Features
- Real-time sensor data collection
- Wireless data transmission (Bluetooth/WiFi)
- Android app for data visualization
- Fish species suggestion based on sensor values

##  Application
The Android application displays sensor values and suggests fish species based on environmental conditions.

##  Project Type
Academic Project – Final Year Project (PFE)
